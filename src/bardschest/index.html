Greetings traveler! You have stumbled upon the wreckage of a caravan plundered by savages. Nearby the body of a wealthy looking bard you find a chest but there is no key in sight. Perhaps there is another way to get the treasure within...
  • Use the left and right arrow keys or drag with the mouse/touch to rotate the lock
  • Catch sound waves in the right sequence to unlock the chest
  • Playing an incorrect note removes one note from the unlock sequence
Bard's Chest is A Web Audio API game made for the 2018 itch.io music jam.

The source code can be viewed on github here https://github.com/n33kos/BardsChest.

Graphics made using Aseprite. Go support this program, it's the best pixel art suite I've found yet!

Music made in good ol’ Garageband.

Open source font from Google Fonts

Hi, I'm a software engineer and independent game developer working out of Denver Colorado. All music and graphics for this project were made by me. Thanks for playing!
